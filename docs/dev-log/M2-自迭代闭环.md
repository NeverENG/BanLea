# M2 · 自迭代闭环（阶段记录）

> 日期：2026-06-09 ｜ 对应计划 §13 M2
> 状态：**已开始**。M1 已提供画像 schema、harness 内核、仓储、触发器与触发式更新入口；M2 开始把真实用户行为接入证据流。

## 本阶段目标（计划 M2）
证据采集 + 触发器 + 重评估 + 版本演化 + 可解释变化，跑通 §5 自迭代闭环。

## 已完成增量一：证据采集服务
- 实现 `createEvidenceCollector`：
  - `recordChat`：记录用户/助手对话证据；
  - `recordSelfReport`：记录用户自评证据；
  - `recordQuiz`：记录测验结果证据；
  - `recordReading`：记录阅读/书单行为证据；
  - `recordRecommendationClick`：记录推荐点击与停留证据；
  - `recordRecommendationSkip`：记录推荐跳过证据。
- 采集服务统一生成 `NewEvidence` 并写入 `EvidenceRepository`：
  - 自动填充 `createdAt`；
  - 保留人类可读 `summary`；
  - 将结构化细节写入 `payload`；
  - 支持调用方覆盖 summary。
- 新增采集服务单测，覆盖六类证据事件。

## 已完成增量二：采集后按需更新入口
- 实现 `recordEvidenceAndMaybeUpdate`：
  - 先将一条标准 `NewEvidence` 写入 `EvidenceRepository`；
  - 随后调用 `runHarnessUpdateIfTriggered`；
  - 若触发条件未满足，则不调用 LLM、不写画像版本、不消费证据；
  - 若触发条件满足，则完成画像生成/局部重评估、版本落库和证据消费；
  - 返回已写入证据与画像更新结果，便于 UI 展示或后台任务记录。
- 新增端到端入口单测：
  - 写入证据后触发条件未满足；
  - 写入证据后首次建档触发并消费证据。

## 已完成增量三：学习事件服务层
- 新增 `features/events`：
  - 面向 UI/功能切片暴露 `recordChat`、`recordSelfReport`、`recordQuiz`、`recordReading`、`recordRecommendationClick`、`recordRecommendationSkip`；
  - 内部先调用 evidence 采集器写入证据；
  - 再调用 `runHarnessUpdateIfTriggered` 按需更新画像；
  - 根据 `domain === "global"` 自动选择主画像，其余 domain 自动选择子画像；
  - 返回已写入证据和画像更新结果，便于界面显示“已记录/已触发画像更新/继续积累证据”。
- 新增学习事件服务单测：
  - 聊天事件未达到触发条件时只记录证据；
  - 推荐点击长停留触发画像重评估；
  - global 自评事件走主画像更新。

## 已完成增量四：开发期事件采集面板
- 将首页从脚手架占位替换为 M2 事件采集面板：

## 已完成增量十：矛盾信号触发策略
- 扩展触发器：
  - 新增 `contradiction_signal` 触发原因；
  - 当未消费证据里同时存在低测验分数与高自评/高信心聊天表达时，触发局部重评估；
  - 矛盾信号优先于普通证据数量阈值，便于及时修正“我以为我会，但测验暴露不会”的画像偏差。
- 扩展触发策略配置：
  - `highSelfReportScore`：结构化自评达到该值时视为高信心；
  - `contradictionScoreGap`：自评分数与测验分数差距达到该值时视为明显矛盾；
  - 默认值分别为 `0.8` 与 `0.3`。
- 扩展 self_report evidence：
  - `SelfReportEvidenceEvent` 支持 `confidenceScore`；
  - 开发期调试面板在记录自评时，会把“测验得分”输入作为自评信心分数写入 payload，方便手动验证矛盾触发。
- 文本信号边界：
  - 用户聊天或自评里出现“掌握 / 懂了 / 会了 / 没问题 / confident”等高信心表达，且同时有低测验分数时触发；
  - 含“不懂 / 不会 / 没掌握 / not confident”等否定表达时不会误判为高信心。
- 新增触发器和状态单测：
  - 高自评 + 低测验触发 `contradiction_signal`；
  - 自信聊天 + 低测验触发 `contradiction_signal`；
  - 分数差距不足时回落为 `low_quiz_score`；
  - 画像状态展示能读出 `contradiction_signal`。
