## 阻塞项（原生构建依赖）
- **Rust**：用 `rustup-init -y --profile minimal` 安装中（winget 直装会因非交互被杀，已改直下 rustup-init）。
- **⚠️ MSVC C++ Build Tools 缺失**：rustup 安装时报 `installing msvc toolchain without its prerequisites`。Tauri 在 Windows 用 MSVC 工具链，**缺少 C++ 链接器则 `cargo build` 会在链接阶段失败**。需安装 **Visual Studio Build Tools（C++ 工作负载）**，约数 GB。
  - 安装方式（待用户确认）：`winget install Microsoft.VisualStudio.2022.BuildTools --override "--quiet --add Microsoft.VisualStudio.Workload.VCTools"`
- PATH：rustup 装好后 `~/.cargo/bin` 需新开终端才进 PATH；本会话内用全路径调用。
- WebView2：Win11 一般已预装（待 `tauri init` 后验证）。
