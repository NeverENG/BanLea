## 构建依赖解决经过（踩坑记录）
1. 本机原装无 Rust、无 MSVC C++ Build Tools。
2. winget 直装 rustup 在非交互后台被杀（退出码 0xC000013A）→ 改直下 `rustup-init.exe -y` 安装。
3. winget 成功装上 **VS 2022 C++ Build Tools（VCTools 工作负载）**。
4. 误用 `rustup ... --force-non-host` 把工具链元数据搞坏（"rustc not applicable / Missing manifest"）→ `rustup toolchain uninstall stable && install stable`（默认 profile）干净重装修复。
5. 最小 crate 与完整 Tauri 应用均编译+链接通过。
