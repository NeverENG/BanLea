## 本次已完成
- 实现 `core/harness` 内核：
  - 主/子画像维度集合选择；
  - 初始画像生成入口；
  - 证据类型到触发维度的确定性推断；
  - 局部重评估 patch schema；

  - scope/domain 与跨层维度越界校验；
  - LLM 调用通过 `HarnessModel` 注入，便于无 API Key 的单测。
- 新增 harness 单测：
  - 主/子维度选择；
  - prompt 只暴露当前层允许维度；
  - quiz / 推荐点击等证据触发维度；
  - patch 只覆盖触发维度，其余沿用旧版；
  - 非法跨层维度拒绝；
  - 无证据时不调用模型、不产生新版本；
  - mock LLM 输出合并路径。
