### 本项目目标 
构建一个个人专属的学习帮助，作用是帮助个人学习，绘制个人学习画像，harness自迭代用户对应学习画像，实时了解用户知道用户的学习底细，用更温和更符合用户学习习惯的方式，帮助用户更好的学习
### 项目粗略要求
1. 能够分类，除了用户主要的 信息，比如喜欢用什么方式学习，哪种方式更好，还有就是每个对应的地方维护不同的 harness 比如用户对物理的掌握程度是怎么样的
2. 所有的harness 都需要从多个维度来分析用户的画像，比如思维，比如学习进度，比如学习能力评估，喜欢什么方式学习，这些都是由你来自评估学习画像
3.能接外部 api ，比如知乎 api 推荐对应的学习资料啥的，很重要
4.简洁清洗的前端，要有设计感，要好看，可以参考一下 gemini vogar的 UI 安装设计感 skill
5.基于 TS，接对应 api，并且能自己上 github 找优秀的代码
### 约束
不要揣摩功能，不懂的就问，然后尽可能的优秀
